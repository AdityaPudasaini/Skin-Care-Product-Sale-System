# Skin Care Product Sale System

A Python-based console application for managing skincare product sales.

## Features
- View available products
- Purchase products
- Update stock
- File handling using modular programming
- Sales invoice generation

## Technologies Used
- Python
- File Handling
- Modular Programming

## How to Run
1. Clone the repository
2. Open terminal inside project folder
3. Run:

python main.py

## Project Structure
- main.py → Main program
- operations.py → Business logic
- read.py → Read product data
- write.py → Write updated data
- data/products.txt → Product database
